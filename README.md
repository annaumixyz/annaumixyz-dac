# ✦ Installation

## Clone Repository

```bash
git clone https://github.com/annaumixyz/annaumixyz-dac.git
cd annaumixyz-dac
```

## Install Dependencies

```bash
npm install
```

---

# ✦ Configuration

Edit:

```bash
config/accounts.json
```

Example:

```json
{
  "default": "main",
  "accounts": {
    "main": {
      "privateKey": "YOUR_PRIVATE_KEY"
    }
  }
}
```

---

# ✦ Launch

```bash
npm start
```

or

```bash
node src/index.js
```
